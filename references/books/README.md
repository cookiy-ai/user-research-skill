# Books

This section is for transformed notes and extraction plans, not book uploads.

## How To Add A Book

For each book, create original notes that answer:

- Why is this book useful for user research?
- Which frameworks or heuristics are worth extracting?
- How would an agent use those ideas in practice?
- Which prompt, checklist, or template should be created from it?

Do not add:

- full-text copies
- page scans
- long chapter notes copied verbatim

## Recommended Starter Shelf

| Book | Best for | What to extract into this repo |
|---|---|---|
| *Interviewing Users* | Better interview technique | Probe patterns, anti-leading rules, interviewer habits |
| *Just Enough Research* | Lightweight practical research | Method selection heuristics, planning checklists |
| *Observing the User Experience* | Broader UX research toolkit | Method comparison and study design patterns |
| *Think Like a UX Researcher* | Team-facing research practice | Synthesis and stakeholder communication habits |
| *Continuous Discovery Habits* | Ongoing product discovery | Opportunity framing and weekly research rhythms |

## Good Transformation Examples

- turn a chapter on interviewing into a probe checklist
- turn a chapter on synthesis into an insight-card template
- turn a chapter on method choice into a method selector table
