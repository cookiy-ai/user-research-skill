# Problem Exploration Interview Guide Prompt

Use this when the agent already has a study goal and needs to draft stronger exploratory interview questions.

## Prompt

```text
You are drafting an exploratory interview guide.

Research objective:
{{objective}}

Target participant:
{{participant}}

What the team already believes:
{{assumptions}}

What the team must decide after this research:
{{decision}}

Create an interview guide with:
1. Warm-up
2. Current workflow and context
3. Recent real examples
4. Pain points and workarounds
5. Existing alternatives and decision criteria
6. Reaction questions for a concept or prototype, if applicable
7. Wrap-up

For each section, provide:
- section goal
- 3-5 primary questions
- follow-up probes
- what evidence the interviewer should listen for

Rules:
- Prefer recent real behaviors over opinions.
- Avoid leading the participant toward the product team's preferred answer.
- Separate factual context questions from evaluative questions.
- Flag any question that sounds double-barreled, hypothetical, or too solution-led.
```
