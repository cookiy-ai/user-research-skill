# Prompt Library

These prompts are written for practical agent use. They are not generic inspiration snippets.

## Sections

- `study-briefs/`
  - Turn a vague product or founder question into a strong study brief.
  - Includes survey instrument design prompts.
- `interview-guides/`
  - Draft sharper interview questions and section structure.
  - Includes JTBD switch interviews, probing technique menus, and usability test scripts.
- `synthesis/`
  - Convert raw notes and transcripts into findings with evidence.
  - Includes qualitative coding, affinity mapping, and opportunity-solution trees.
- `stakeholder-readouts/`
  - Turn findings into founder-, PM-, or design-facing summaries.
  - Includes data-story narratives for mixed qualitative and quantitative evidence.
- `recruitment/`
  - Outreach copy and screening-call scripts for participant recruitment.

## Usage Rule

Use these prompt files as building blocks:

- fill in the placeholders
- keep evidence and outputs structured
- prefer concrete constraints over style fluff

If a prompt produces a good repeated workflow, consider turning it into a future installable skill after compatibility review.
